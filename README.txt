BTIS Survey App v0.4

Changes:
- Traveller ID is now auto-generated.
- Traveller Account layout changed:
  Row 1: Name/Nickname + Nationality
  Row 2: Guide dropdown + Traveller ID
- If the same traveller name and nationality already exist, the existing account opens.
- Guide dropdown uses registered guides.
- New guides can be registered from the account page or System Settings.
- Passport / National ID is required when registering a guide.
- Previous v0.3 functions remain: reopening surveys, rating comments, print draft, and history logging.

Open index.html in a browser.


BTIS Survey App v0.5

Changes:
- Optional comment is no longer lost when changing a rating.
- After selecting a rating, the cursor automatically moves to the optional comment field.
- Input Mode added: Guest / Guide-BTIS.
- In Guest mode, the app skips FOR BTIS USE ONLY and goes directly to Optional Contact.
